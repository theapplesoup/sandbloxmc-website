FAQ

Can I join with Minecraft Bedrock?                                 

Unfortunately, SandbloxMC is only available on the Java Edition of Minecraft.

How do I travel to The End dimension?                                 

The End dimension is disabled on SandbloxMC. To craft blocks exclusive to this dimension, please see the Information page.

Why can't I create a player shop? ("Integration Towny denied the shop-creation")

Player shops must be created on townblocks set to "shop". To set a plot type to "shop", use the command /plot set shop on the chunk where you'd like to create a shop.

Why can't I create a nation? ("The town can't afford to start a nation, which costs 500.0")

To create a nation, you must have a minimum of 500 gold stored in your town's bank. To deposit gold into your town's bank, use the command /t deposit ($).

Need more assistance?

Please create a support ticket on our official Discord server.
