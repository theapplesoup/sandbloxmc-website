Towns & nations

SandbloxMC uses the Towny plugin to manage land claiming and town creation. This page provides you with basic information about the plugin to get you started. For a full list of commands, visit the Towny website here.

Joining a town

  • Ask to join a town in chat, then use the /accept command to accept a player's invite.
  • Type /t spawn to teleport to your town's spawn.
  • You can use the generic /town command to see information about the town you reside in.

Creating a new town

  • To create a new town where you're standing, type /town new (town name). (This costs 64 gold)
  • /t invite (player) invites a player to your town.
  • /t claim claims the chunk you are standing in. (This costs 16 gold)
    • Note: You can only claim chunks adjacent to your town

Joining a nation

  • /n join (nation name) allows you to join an open nation.
    • Note: To check if a nation is open use /n (nation name).
  • /t invite accept (nation name) allows you to accept a nation invite.

Creating a new nation

  • Creating a nation costs 1,000 gold. You must deposit this amount into your town bank with the command /t deposit (amount)
  • To create a new nation, type /n new (nation name).
  • You can invite other towns to your nation with /n add (town name).
  • You can add nation allys with /n ally add (nation name).
