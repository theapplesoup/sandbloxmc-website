Joining the server

Before joining the server, make sure you have version 1.20.4 of Minecraft: Java Edition installed on your computer. Follow these steps to connect to the server:

  1. Lauch Minecraft: Java Edition
  2. Click on the 'Multiplayer' button
  3. Choose the 'Add server' option
  4. Rename the server to "SandbloxMC" (optional)
  5. Under server address, type sandbloxmc.com
  6. Click 'Done'
  7. Select SandbloxMC, then click 'Join Server'
