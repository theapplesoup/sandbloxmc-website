Introduction

SandbloxMC is a Minecraft: Java Edition multiplayer server on a 1:3000 scale map of planet earth. The server uses the Towny plugin to allow players to claim land, form towns and establish nations.

To view a live Google Earth style map of the world, click on the link here.

Objective

On SandbloxMC, our players create their own objectives. Whether you're interested in creating a business empire, building the largest nation or becoming the deadliest bounty hunter, this server is a sandbox for you to play however you'd like.

If you're still stuck, here's a list of common objectives:

  • Building a town/nation
  • Starting an item shop
  • Creating and selling map art
  • Purchasing real estate
